파이썬에서는 논리연산자 할때 단락 평가(앞이 맞으면 뒤 값을 보지도않음)에 따라 반환값이 결정된다.
첫번째 값까지만 봐도 된다면 첫번째 값을 반환하고, 두번째 값까지 판단해야한다면 두번째 값이 반환된다. 이게 바로 단락 평가이다.
True and 'abc'  #'abc'
'abc' and True  #True
True or 'abc'  #True
'abc' or True  #'abc'
False and 'abc'  #False
0 and 'abc'  #0  0은 False이므로 and 연산자는 두번째 값을 평가하지 않음.
